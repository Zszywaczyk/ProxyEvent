# ProxyEvent

Event management system - my first PHP project. University assignment (2018).

Users can create accounts (organizer/participant/owner), create events with locations shown on Google Maps, and manage participation.

## Features

- User registration with role selection (organizer, participant, owner)
- Event creation with location on Google Maps
- Space/venue management with GPS coordinates
- Session-based authentication
- MySQL database backend

## Tech

- PHP (vanilla, no framework)
- MySQL via MySQLi
- Google Maps JavaScript API
- CSS (custom, no framework)
- Sessions for auth
